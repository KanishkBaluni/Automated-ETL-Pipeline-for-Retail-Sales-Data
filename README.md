# Automated-ETL-Pipeline-for-Retail-Sales-Data

Description:-
Automated ETL pipeline built using Python and SQL to process daily retail sales data. The workflow performs data extraction, cleaning, validation, and transformation before loading structured datasets for reporting.

Tools:- Python (Pandas, NumPy), SQL, Power BI

Work Performed:-

Built an ETL workflow for daily sales data.
Implemented automated data cleaning and validation.
Reduced manual data preparation by ~70%.
Integrated transformed data with Power BI dashboards.
Outcome: Improved reporting efficiency and data reliability for sales and inventory analysis.
